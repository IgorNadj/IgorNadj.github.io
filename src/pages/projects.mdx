### Next Season
Next Season is a little website which tells you when your favourite shows are returning.

Built with [Parser Expression Grammer](https://en.wikipedia.org/wiki/Parsing_expression_grammar) to extract relevant information from the semi-structured files the IMDB API provides. 
See project on [GitHub](https://github.com/IgorNadj/nextseason-web).
